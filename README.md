# Internship Tracker [In Progress]

A full-stack application to help students track job applications with email analysis features.

## Tech Stack

### Backend
- Java 17 + Spring Boot 3.3.1
- Spring Security + JWT for authentication
- Spring Data JPA + Hibernate
- PostgreSQL
- Maven
- ChatGPT API for analyzing emails

### Frontend
- React
- Axios for API calls
- Chart.js/Recharts for analytics
- CSS for UI/UX
